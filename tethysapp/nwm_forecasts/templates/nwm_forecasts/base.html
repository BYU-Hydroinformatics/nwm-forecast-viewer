National Water Model Forecast Viewer Guide

    About the App

    Users can obtain forecasts of different hydrologic variables available from any of the associated National Hydrography Datasets (e.g., channel, rivers, land surface grids, reservoir).

    How to Use the App
  1. Configuration - Select a forecast range: Short, Medium, Long, or Analyis & Assimilation
  2. Geometry - Select a geometry: Channel, Land, or Reservoir
  3. Variable - Select a variable: This list will change depending on the type of geometry
  4. COMID - Click a feature on the map: Clicking on a feature will populate this field (you may need to zoom in to be able to see some features). If you know the ID you can enter it directly
  5. Date and Time - Select a date: The date and time options will vary depending on the forecast type
  6. View Forecast - Click submit to view the forecast
Developed at the BYU Hydroinformatics Lab
